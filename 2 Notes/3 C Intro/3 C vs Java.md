Assuming to the people who've taken CS61B, C isn't that far from Java, as Java is developed from C in the first place.

![[Screenshot 2026-02-06 at 9.19.40 PM.png | 500]]

Just by looking at the image, we can see some differences in the 2 languages
- C is focused on functions, Java is focused on classes
- C uses `gcc` to compile, Java uses `javac` (java-c?)
- C's storage of memory is manually managed, Java has garbage collection

![[Screenshot 2026-02-06 at 9.24.07 PM.png|500]]

This... is just a bit more on the subtle stuff, like comments, constants... just conventions, the modern C (C99 and beyond) acts the same as Java.
