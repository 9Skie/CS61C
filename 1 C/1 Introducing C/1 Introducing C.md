Hi, these series of notes is intended to work alongside CS61C to learn about computer architecture, and is learning about the content of the K&R C Programming Language book.

Simply, C was a programming language developed in the early 1970s at Bell Laboratories.

It was a by-product of the Unix operating system, as Unix was originally written in assembly, and this was really painful for developers. 

Thompson first developed B, but that wasn't enough for UNIX, and so Ritchie extended on it and developed C, which allowed Unix to be entirely written in C.

C influenced a lot of modern programming languages, like:
- C++, which is basically C with OOP
- Java, developed from C++, with the idea of "write once, run anywhere"

This was the time when computer science was transitioning from thinking from a hardware level, to writing portable and efficient software.

Operating systems like Unix and Linux are written in C, early networking implementations were written in C... and so, you should learn it!
